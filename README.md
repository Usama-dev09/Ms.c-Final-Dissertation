# AI-Driven Health App Recommendation Analysis

This repository contains the code, data references, and supporting materials for a thesis project analyzing how data- and AI-driven recommendation features in health applications influence diet and physical activity behaviors among young adults in the United States.

The study relies on secondary, publicly available datasets and applies machine learning and statistical techniques to examine user engagement, adherence, and long-term behavior trends.
